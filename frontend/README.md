# Campaign Validator - Frontend

## Build para Produção

Este diretório contém o código-fonte do frontend.

### Comandos

```bash
# Desenvolvimento
npm run dev

# Build para produção
npm run build

# Preview do build
npm run preview
```

### Deploy

Após o build, os arquivos estarão em `dist/`. Faça upload desses arquivos para sua hospedagem compartilhada.

**Importante:** Certifique-se de que o arquivo `.htaccess` está na pasta `dist/` antes do upload!
